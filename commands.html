The lamb command

lamb> 命令详细列表，以及该命令支持的系统版本和功能说明

help1.2显示所有 lamb 命令帮助信息
show mt1.2显示核心 MT 上行消息队列统计信息
show mo1.2显示核心 MO 下行消息队列统计信息
show core1.2显示核心模块的运行状态信息
show client1.2显示当前 cmpp 客户端连接信息
show server1.2显示所有 server 业务处理进程信息
show account1.2显示所有 account 账户信息
show channel1.2显示运营商 channel 网关状态信息
show gateway1.2显示运营商网关账户信息
show delivery1.2显示下行推送路由表信息
show routing1.2显示账号的上行路由表信息
show log1.2查看模块日志输出信息
kill client1.2强制断开一个 cmpp 客户端连接
kill server1.2强制一个 server 业务进程退出
kill channel1.2强制一个 channel 网关退出下线
start server1.2启动一个 server 业务处理进程
start channel1.2启动一个 channel 网关通道连接
change password1.2修改 lamb 管理平台的用户密码
show version1.2显示 lamb 系统的版本信息
exit1.2退出 lamb 系统登录
