kill client <id> 强制断开一个 cmpp 客户端连接

参数 id 为客户端账户ID编号
使用示例  查看当前登录的客户端 
lamb> show client
Id User   Company   Host            Status Speed Error
----------------------------------------------------------
1  901234 1         192.168.1.254     ok   10    0
2  901235 1         192.168.1.247     ok   10    0
使用示例   强制断开一个 cmpp 客户端连接 
lamb> kill client 1

当断开一个 client 客户端连接后，客户端可能很快再次连接
