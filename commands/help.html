help 查看所有命令帮助信息

help                        Display help information
show mt                     Display mt queue information
show mo                     Display mo queue information
show core                   Display core module status information
show client                 Display client connection information
show server                 Display service module status information
show account                Display account information
show channel                Display carrier channel status information
show gateway                Display carrier gateway information
show delivery               Display downlink routing information
show routing <id>           Display uplink routing information
show log <type>             Display module log information
kill client <id>            Kill a client disconnected
kill server <id>            Kill a service processing module
kill channel <id>           Kill a gateway disconnected
start server <id>           Start a service processing module
start channel <id>          Start a gateway channel service
change password <password>  Change user login password
show version                Display software version information
exit                        Exit system login
