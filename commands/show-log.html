show log <type> 查看所有 server 业务处理进程信息

使用示例1  参数 type 表示日志类型，使用 ? 问号显示可用的所有类型列表 
lamb> show log ?
 
client       Client log information
server       Server log information
channel      Channel log information
mt           MT module log information
mo           MO module log information
scheduler    Scheduler log information
delivery     Delivery log information
testd        Test module log information
daemon       Daemon module log information
database     Database log information
使用示例2  查看 channel 运营商通道日志信息 
lamb> show log channel
 
May  9 14:25:41 iKbp173nglzx9jagu7jxjrZ lamb-gateway[26191]: can't connect to server 123.55.180.155
May  9 14:26:16 iKbp173nglzx9jagu7jxjrZ lamb-gateway[26216]: can't connect to server 123.55.180.155
