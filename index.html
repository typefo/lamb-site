Lamb Gateway

一个简单易用的，基于模块化设计的开源短信网关平台

A simple and easy to use, open source SMS gateway platform based on modular design

开放源代码

lamb 基于 GPL v3.0 开源协议，您可以获取源代码，没有任何功能限制的自由使用，并且基于 C 语言开发。如果没有你想要的功能，你可以随心所欲的去改造它

模块化设计

一开始就采用模块化的设计，业务模块之间隔离，保证平台稳定性。模块间采用 Google Protocol Buffer 和 Nanomsg 作为内部的高效通信协议库
